
# GrievX – AI-Powered Public Grievance Resolution System

## Overview

GrievX is a first AI-powered public grievance resolution platform designed to help citizens report and track public issues quickly and efficiently.

The platform allows citizens to submit complaints using:
- Text
- Voice
- Images
- Videos

without requiring English literacy or technical knowledge.

GrievX automatically:
- Classifies complaints using AI
- Detects responsible authorities
- Predicts complaint resolution time
- Tracks complaint progress
- Escalates unresolved issues automatically

The system focuses on transparency, accountability, and faster grievance resolution across Tamil Nadu governance.

---

# Problem Statement

In Tamil Nadu, many public issues such as:
- Broken roads
- Water shortages
- Garbage overflow
- Drainage problems
- Power cuts

remain unresolved for months because:
- Citizens do not know whom to contact
- Complaints are not tracked properly
- Existing portals are difficult for rural citizens
- No structured escalation mechanism exists

GrievX solves this problem using AI-powered complaint management and automated escalation workflows.

## AI-Based Resolution Time Prediction

The system predicts:
- Estimated resolution time
- Delay probability
- Escalation risk
- Complaint priority

### Example

```text
Complaint:
"Garbage overflow near bus stand"

AI Prediction:
Estimated Resolution Time: 3 Days
Priority Level: Medium
Escalation Risk: Low
```

This prediction is generated using:
- Historical grievance data
- Complaint category
- Area/District data
- Authority response history
- Severity level
